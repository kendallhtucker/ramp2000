# Ramp.com — Y2K Edition 💾✨

A maximalist, early-2000s reimagining of [ramp.com](https://ramp.com). Built because the modern web has become too minimalist.

🌐 **Live site:** [kendall-tucker-ramp.github.io/ramp2000](https://kendall-tucker-ramp.github.io/ramp2000/)

## Features

- 🌈 Rainbow gradient logo with `<blink>` and `<marquee>` tags
- 💿 Hit counter + sparkle cursor trail
- 📨 "You've Got Mail" popup
- 🏆 Web awards + browser badges ("Best viewed in Netscape Navigator 4")
- 🎵 Member Login, guestbook, B2B SaaS Webring™
- 📺 Banner ads (FREE iPod, Punch the Monkey)
- 🤝 All buttons link to real Ramp via referral

## Tech

Pure HTML / CSS / vanilla JS. No build step. Deliberately uses tables for layout, `<font>` tags, and animated GIF aesthetics.
